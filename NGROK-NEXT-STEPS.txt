TOCSEA – Ngrok is almost ready. Do these 3 steps:

1) ADD YOUR NGROK AUTHTOKEN (one-time)
   Get it from: https://dashboard.ngrok.com/get-started/your-authtoken
   In PowerShell run:
   ngrok config add-authtoken YOUR_TOKEN_HERE

2) START NGROK (new terminal)
   Laravel is already running on port 8000. In a NEW terminal:
   cd c:\projects\tocsea_system
   .\start-ngrok.ps1
   Or: ngrok http 8000
   Copy the HTTPS URL (e.g. https://abc123.ngrok-free.app).

3) UPDATE .ENV AND RESTART LARAVEL
   In .env set:
   APP_URL=https://YOUR-COPIED-NGROK-URL.ngrok-free.app
   SESSION_SECURE_COOKIE=true
   Then run:
   php artisan config:clear
   php artisan cache:clear
   Restart the Laravel server (Ctrl+C in that terminal, then run again):
   php artisan serve --host=0.0.0.0 --port=8000

Then send the HTTPS ngrok URL to your client.
